# Swiggy Sales Dashboard

## Project Overview
This project analyzes Swiggy food delivery data to understand revenue, orders, customer ratings, and city performance.

## Dashboard Insights
- Total Revenue: ₹6,88,835
- Total Orders: 1000
- Average Rating: 3.98
- Delivery Success Rate: 84.5%

## Key Features
- Orders trend over time
- Top restaurants by revenue
- Revenue by city
- Orders vs Revenue comparison
- Food category distribution
- Monthly revenue trend
